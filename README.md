# 📌 Piyush Raikwar – Personal Portfolio

This is my personal portfolio website built using HTML and CSS. It showcases my skills, projects, certifications, and provides a way to connect with me.

---

## 🚀 Features

* 🔹 **Responsive Navigation Bar** with smooth scrolling
* 🔹 **Hero Section** with introduction and profile
* 🔹 **Skills Section** highlighting programming, data, and tools
* 🔹 **Projects Section** with live project links
* 🔹 **Certificates Section** in grid layout
* 🔹 **Contact Section** with social media links
* 🔹 **Working Contact Form** using Formspree
* 🔹 **Hover Effects & Animations** for better UI
* 🔹 **Custom Theme using CSS Variables**
* 🔹 **Background Image with Fixed Effect**

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts (Inter)
* Font Awesome Icons
* Formspree (for form handling)
